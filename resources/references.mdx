## Essential Tools

### Development Tools

<Card title="rustup" icon="download" href="https://rustup.rs/">
  The Rust toolchain installer. Manages Rust versions, targets, and components.
  
  ```bash
  # Install rustup
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  
  # Update Rust
  rustup update
  
  # Check version
  rustc --version
  ```
</Card>

<Card title="cargo" icon="box" href="https://doc.rust-lang.org/cargo/">
  Rust's build system and package manager. Included with rustup.
  
  ```bash
  # Create new project
  cargo new my_project
  
  # Build project
  cargo build
  
  # Run project
  cargo run
  
  # Run tests
  cargo test
  ```
</Card>

### Code Quality Tools

<CardGroup cols={2}>
  <Card title="Clippy" href="https://github.com/rust-lang/rust-clippy">
    A linter to catch common mistakes and improve your Rust code.
    
    ```bash
    # Install
    rustup component add clippy
    
    # Run
    cargo clippy
    ```
  </Card>
  
  <Card title="rustfmt" href="https://github.com/rust-lang/rustfmt">
    Automatic code formatter following Rust style guidelines.
    
    ```bash
    # Install
    rustup component add rustfmt
    
    # Format code
    cargo fmt
    ```
  </Card>
</CardGroup>

### Documentation Tools

<Card title="cargo doc" href="https://doc.rust-lang.org/rustdoc/">
  Generate documentation for your project and dependencies.
  
  ```bash
  # Generate and open documentation
  cargo doc --open
  
  # Include private items
  cargo doc --document-private-items
  ```
</Card>

## Package Registry

<Card title="crates.io" icon="box" href="https://crates.io/">
  The official Rust package registry. Search for libraries (called "crates") to use in your projects.
  
  **Popular crates:**
  - `serde` - Serialization/deserialization framework
  - `tokio` - Async runtime for I/O operations
  - `clap` - Command-line argument parser
  - `reqwest` - HTTP client
  - `axum` - Web framework
